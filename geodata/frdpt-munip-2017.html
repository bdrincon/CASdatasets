fredpt17R Documentation

French geospatial dataset for departments and cities

Description

fredpt17 is a spatial database containing geospatial information of French departments to be used with pricing actuarial games' files, see pricingame.

fremuni17 is a spatial database containing geospatial information of French cities (known as communes) to be used with pricing actuarial games' files, see pricingame.

Usage

data(fredpt17)
data(fremuni17)

Format

fredpt17 is of class sf. fremuni17 is of class sf.

Source

See https://actinfo.hypotheses.org/86 for the third pricing game.

See Also

See the sf class.

Examples


# (1) load of data
#
data(fredpt17)
data(fremuni17)
